1、吴恩达机器学习系列课程

1. 什么是机器学习？

1.1 在没有明确设置的情况下，使计算机具有学习能力的研究领域。——Arthur Samuel

1.2 A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E. ——Tom Mitchell

译：计算机程序从经验E中学习，解决某一任务T进行性能度量P，通过P测定在T上的表现因经验E而提高。

2.监督学习(Supervised learning )

2.1 定义：根据已有的数据集，知道输入与输出结果之间的关系，根据这种已知的关系，训练出一个最优的模型。通俗一点，可以把机器学习理解为我们教机器如何做事情。

2.2 分类：回归（Regression）,分类（Classification）

2.2.1 回归（Regression）

回归是针对于连续型变量的。

2.2.2 分类（Classification）

分类是针对离散型的，输出的结果是有限的。

3.无监督学习(Unsupervised learning )

3.1 定义： 我们不知道数据集中数据、特征之间的关系，而是要根据聚类或一定的模型得到数据之间的关系。

可以这样理解，比起监督学习，无监督学习更像是自学，让机器学会自己做事情，是没有标签的

即给定一些数据，找到数据之间的关系，根据特征聚类等，对数据进行划分类别
